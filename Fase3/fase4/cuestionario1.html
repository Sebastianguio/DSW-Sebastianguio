    Sitios Responsivos con Media Queries Son aquellos que nos sirven para definir estilos, son herramientas fundamentales para implementar y diseñar páginas con mayores estilos y formas convirtiéndose en un gran aliado fundamental del cualquier diseñador, Media Queries ha sido una evolución de CCS ya que Media Queries suponde la primera estructura condicional del lenguaje y también otra gran ventaja es que se adapta a disintos tipos de pantalla Operadores lógicos para las Media Queries Para combinar diversas condiciones podemos usar los operadores lógicos, de una manera similar a como se usan en lenguajes de programación. Los que tenemos disponibles son: Operador and: las dos condiciones deben cumplirse para que se evalúe como verdadera. Operador not: es una negación de una condición. Cuando esa condición no se cumpla se aplicarán las media queries. Operador only: se aplican las reglas solo en el caso que se cunpla cierta circunstancia. Operador or: no existe como tal, pero puedes poner varias condiciones separadas por comas y cuando se cumpla cualquiera de ellas, se aplicarán los estilos de las media queries.
