    Etiquetas HTML

    Las etiquetas son la estrutura basica de HTML. Estas tiene dos propiedas basicas: atributos y contenidos, vemos que cada atributo y contenidos tiene ciertas restricciones para que podamos considerar valido el documento HTLM, todo elemento tiene una etiqueta de inicio y una de cierre.

    Valores de un atributo

    Se refiere cuando se coloca un atributo por lo general debemos colocar un valor, ya que nos ayuda a ser más específicos con lo que queremos mostrar, como una característica específica, o una cantidad, o en qué medida se ejecutar este atributo los valores de loa atributos siempre tiene que ir entre comillas “y específicamente los valore de un atributo se especifican con el signos igual (=)
